# PPSP
Peer to peer streaming protocol (PPSP) is developped by IETF PPSP WG.

PPSP is composed of the following  protocols
* Peer protocol (PPSPP, IETF RFC 7574)
* Tracker protocol (PPSTP, IETF RFC 7846)
